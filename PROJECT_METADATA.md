# Repository Metadata

## Project Name
DNA Sequence Analysis AI Backend

## Short Description
Production-oriented FastAPI backend for FASTQ ingestion, QC analytics, model scoring, report generation, and chatbot-ready responses, including streaming stage updates and public-source FASTQ ingestion (URL/ENA).

## Primary Domain
Bioinformatics / Genetic Data Science

## Core Capabilities
- Upload FASTQ and run sequential analysis pipeline
- Stream stage-by-stage NDJSON responses
- Fetch FASTQ from direct URL or ENA accession
- Return structured JSON outputs:
  - `qc_metrics`
  - `model_results`
  - `analysis_summary`
  - `chatbot_response`
